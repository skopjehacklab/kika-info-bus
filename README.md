# Kika Info Bus 2.0

Based on MQTT (old stuff is in `legacy/`).

## publishers
* temp sensors
* open/closed button(s)
* PIR sensors
* bell door?
* openwrt num associations
* …

## subscribers
* store to influxdb
* send twitter message
* send irc message
* open door
* LEDs in the hacklab
* LED strips in the hacklab
* LCD in the hacklab
* AC (turn off)
* …
